CoRe Planner (Cecill licence)

CoRe Planner is a planner based on a Hieachical Transition Network mechanism. Some sample domains are available in the directory "domains". The API of CoRe Planner is accessible here and the CoRe Planner syntax is available in BNF format.
To launch CoRe Planner from the root of the project use the command:
    java -jar lib/planner.jar \
     -d doc/domains/blocksworld/blocksworld.jap \
     -p doc/domains/blocksworld/bw-pb-10-4.jap

This command will lauch the planner on the blockworld problem.

The command line is:

[Usage Planner [-f;h] -d <domain> -p <problem> -n <number> -w <window> -l <log-level>] :
  -d  the path of the file containing the domain description.
  -p  the path of the file containing the problems description.
  -s  the name of problem to solve. If the name is not specified all problems are solved.
  -n  use this option to express the number of conjectures to find. This number
      must be greater than 1. Use -n all to find all conjectures.
  -f  use this option if problem use negative facts.
  -l  the log level for debuging. The log level parameter is into the interval [0;10].
      0 for indicating no output. The default value is 1.
  -w  the size of the assumption window. The window parameter cannot be negative.
      The default value is 0 (i.e., no assumption must be done).
  -h  Display this help information.
