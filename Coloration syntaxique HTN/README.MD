HTN est un package de coloration syntaxique de GTOHP pour sublime texte basé sur celui de PDDL.
Il est compatible avec les extensions .pddl 

--------------------------------------------------
Instalation : 
	Extraire le package dans le dossier sublime-test-3/Packages/User/
		- Sous mint il est situé dans le dossier .config dans le /home
		- Sinon voir la doc sublime text pour l'instalation de package custom.
